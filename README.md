This site collects documentation and resource links useful for those wishing to test and demonstrate extensions to the Ghidra decompiler.
It is structured as a Hugo + Docsy static web site generator.  Any push to the main branch should result in a Github workflow
rendering the site to [https://thixotropist.github.io/ghidra_decompiler_commons](https://thixotropist.github.io/ghidra_decompiler_commons/).

As of April 2026 this site is very new and mostly accumulating raw material in need of edits.  Comments/Issues/PRs are welcome.
